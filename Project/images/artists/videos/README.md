# Artist Video Files

This directory should contain short video clips showcasing each artist at work.

## Required Video Files:
- jane-doe-painting.mp4 - Jane painting a coastal sunrise scene
- mark-smith-sculpting.mp4 - Mark creating a geometric sculpture
- tims-woodshop-crafting.mp4 - Tim hand-crafting an oak serving board
- sarah-johnson-mountains.mp4 - Sarah painting in the Sierra Nevada mountains
- alex-chen-cityscape.mp4 - Alex capturing urban architecture in San Francisco
- emily-brown-forest.mp4 - Emily painting among the redwoods in Muir Woods

## Video Specifications:
- Format: MP4 (H.264)
- Duration: 10-20 seconds each
- Resolution: 720p or 1080p
- Aspect Ratio: 16:9 or 4:3
- File Size: Keep under 5MB each for web performance

## Accessibility:
- Each video has corresponding WebVTT caption files in the /captions/ subdirectory
- Videos should include natural audio when possible
- Captions describe both spoken content and visual actions
